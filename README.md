ImcCalculator

This repository contains the ImcCalculator project, a React‑based Body Mass Index (BMI) calculator developed to practice front‑end and React fundamentals.

Live version:
https://lucashenrique1908.github.io/ImcCalculator/

Description

ImcCalculator is a single‑page application that enables users to calculate their Body Mass Index (BMI) by entering height and weight. The application processes inputs and displays results in real time with a clean, responsive interface.

This project was developed primarily for learning and skill development with React.

Technologies Used

* JavaScript

* HTML

* CSS

Purpose

The main goals of this project are to:

 - Handle user input and events

 - Improve front‑end interactivity and UI responsiveness

 - The app demonstrates foundational React concepts with a practical implementation.

How to Run Locally

Clone the repository:

git clone https://github.com/lucashenrique1908/ImcCalculator.git


Notes

This project is part of ongoing front‑end learning and showcases the application of React skills in a focused and interactive application.
